(Image of the Overview for the whole process in Power Automate)

Creating new flow

Go to Power Automate https://asia.flow.microsoft.com/en-us/ and create a an Instant Cloud Flow and name the flow as "Email Format"

Choose flow trigger as "Manually trigger a flow"

After creating a flow, click on add input and add "Yes/No"

Get items from sharepoint

Next, we will use "Get Items" to fetch details from an existing Sharepoint List of your choice

Assign items into HTML table

Add a HTML Table and fill in the Header and Values accordingly

Send email with "output" variable as HTML table

Next we will send an email to your desired recipient and attach the HTML Table output into the body of the email.

This is how the final email will look like.
