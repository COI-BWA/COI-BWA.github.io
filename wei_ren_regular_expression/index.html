The use case aims to show a basic example of how to use Plumsail Documents Regular Expression Match to use regular expressions in power automate as well as show regular expressions can be used to filter data.

Setting up Plumsail Account

First, start by going to https://plumsail.com/documents to create an account

(Screenshot of Plumsail website)

After creating an account, click on My Account on the top right. Next, Select documents from the page

Setting up an API Key

Next, Select API Keys on the left and create an API key

(Example of an API key created)

Setting up connection in power automate

To use the functions from Plumsail in power automate, use the API key created to login

Connect name input should be same as the API key name

API key input could be copied from the "Secret" section where you create the API key

(Screenshot of creating a connection to use Plumsail Activities inside power automate)

Setting up list to store data

Set up a microsoft list(for my use case it is Test Regex) and have the following columns (Price and Quantity as single line of text)

(Screenshot of the columns in Microsoft list)

Initializing Variable

First, add a new action Initialize Variable and declare the name as TestString and the type of it as String. After that, copy the text below and paste it into the value

*** Order information ***Time zone: GMT+00:00Reference: 4563452Order number: 1400Payment method: Visa/MasterCardOrder status: Credit card payment authorized*** Ordered products ***Product code: D3F13B23B0Title: TiresPrice: 150Quantity: 4Product code: FD3423DETitle: CookiesPrice: 30Quantity: 20Product code: KFG45GK445Title: ToolsPrice: 50Quantity: 10*** Delivery information ***First name: JohnLast name: DoeCompany: Happy Company LtdCountry: AustraliaPhone: 01 2345 6780

(Screenshot of the inputs for initialize variable action)

Regular Expression Match

Next, create a new action Regular Expression Match from Plumsail Document. In the pattern input, copy and paste the text below

Title: (?<Title>.+|)

Price: (?<Price>.+|)

Quantity: (?<Quantity>\d+)

The regular expression for this is to filter the text above for any title in the text it will take the values after it. The same goes for Price and Quantity as well

Afterwards, in the text input, select TestString as a variable

(Screenshot of the inputs in Regular Expression Match action)

Create Item

Lastly, create a new action called Create Item from Sharepoint and select the list created which stores the input. The list name should be Test Regex.

In the title input, select the Title created from the previous action which is the Plumsail Document Regular Expression Match. For the Price and Quantity, select the Price and Quantity from the previous action as well.

(Screenshot of the inputs in Create Item in Sharepoint)

Note: Do note that when you select the Title as the input for the first time, an apply for each will be created automatically which is normal.
